# Front-end Service

## Overview
The Front-end Service is a crucial part of the solveMyProblem SaaS project, providing the user interface for the application. This service interacts with various backend microservices through API endpoints to fetch and display data to the users. It supports user authentication, submission management, result viewing, credit management, and more.

## Features
- User authentication (login, logout, sign up)
- Manage submissions (create, view, delete)
- View results
- Manage user credits
- Access logs and analytics (admin only)
